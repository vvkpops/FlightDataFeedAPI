Authentication

Include your API key in every request header:

x-api-key: fda_your_api_key_here
GET /api/v1/flights

Retrieve paginated flight data with optional filters.

ParameterTypeDescription
datestringFilter by generation date (YYYY-MM-DD)
from_datestringStart date range (inclusive)
to_datestringEnd date range (inclusive)
callsignstringExact callsign match (e.g. PVL910)
departurestringDeparture ICAO code (e.g. CYUL)
arrivalstringArrival ICAO code
registrationstringAircraft registration (e.g. C-GMXE)
pagenumberPage number (default: 1)
limitnumberResults per page (default: 50, max: 200)

Example:

curl -H "x-api-key: YOUR_KEY" \
  "https://your-host/api/v1/flights?date=2026-03-04&departure=CYUL&limit=10"

Response:

      "arrivaltime": "2026-03-04T09:45:00.000Z",

GET /api/v1/flights/:id

Retrieve a single flight by ID.

curl -H "x-api-key: YOUR_KEY" "https://your-host/api/v1/flights/42"
{
  "data": {
    "id": 42,
    "callsign": "PVL847",
    ...
  }
}
GET /api/v1/dates

List all generation dates and flight counts.

curl -H "x-api-key: YOUR_KEY" "https://your-host/api/v1/dates"
{
  "data": [
    { "generation_date": "2026-03-05", "flights_count": 36, "created_at": "..." },
    { "generation_date": "2026-03-04", "flights_count": 36, "created_at": "..." }
  ]
}
GET /api/v1/stats

Get summary statistics.

curl -H "x-api-key: YOUR_KEY" "https://your-host/api/v1/stats"
{
  "data": {
    "total_flights": 180,
    "total_days": 5,
    "latest_generation_date": "2026-03-05"
  }
}
